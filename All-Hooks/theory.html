All Important Hooks in React

What are Hooks?

Hooks are the functions to use some react feature in functional components.
In other words, Hooks are functions that make Functional Components work like Class Components.

  1. useState : UseState Hook is a function to add state in functional component.
    What is State ? State is nothing but just values or variables of your component.
  2. useEffect: UseEffect is used to perform side effects in our components.
    What is Side Effect ? Side effects are actions which are performed with the "OUTSIDE WORLD"
    We perform a side effect when we need to reach outside of our React component to do something.
    eg: Fetching data from an API, Updates the DOM document and window, Timer functions setTimeout & setInterval, are most common side effects.

  5. useRef : UseRef allows us to access the DOM elements, for creating mutable variables which will not re-render the component.
